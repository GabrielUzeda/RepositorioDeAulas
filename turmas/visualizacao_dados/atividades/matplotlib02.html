  4. Questão 4 — Leitura de Código: plt.text()

    O que o código abaixo fará? Descreva onde, visualmente, o texto "Pico de Vendas" aparecerá no gráfico.

meses = [1, 2, 3, 4, 5]
vendas = [100, 120, 180, 150, 140]
plt.plot(meses, vendas)
plt.text(3, 180, '  Pico de Vendas')
plt.show()
